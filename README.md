## Rock-Paper-Scissors
This is an interactive console game created as part of The Odin Project's Foundations Course.

## Objectives
The primary goals of this project were to:

- Practice Problem Solving: Apply the three-step process (Understand, Plan, Divide and Conquer) learned in the problem-solving lesson.
- Integrate Core JavaScript: Utilize functions, conditional logic (if/else), loops (for loops), and user input handling.
- Develop Game Logic: Implement the complete game rules, including computer choice generation, win/loss determination, and score tracking over multiple rounds.

##  How to Play

- Open the file in your web browser.
- Open the Developer Console (usually F12 or Cmd/Ctrl + Shift + J).
- The game will begin automatically, prompting you for your choice.
- Enter your selection: rock, paper, or scissors.
